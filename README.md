# Projeto de Casamento

Este é um projeto de um aplicativo web para o casamento de Millena e Robert. O aplicativo permite que os convidados inscrevam-se e reservem presentes.

## Tecnologias utilizadas

- Backend: Node.js, Express, Prisma, TypeScript
- Frontend: React, Vite, TypeScript, Tailwind CSS
